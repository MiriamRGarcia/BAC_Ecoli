AMIGO_RIdent -- Robust identification of model unknowns

Contents

  • See also
  • Example
  • Performs a robust analysis of practical identifiability by computing robust confidence regions or expected uncertainty for the parameters using a Monte-Carlo based approach. This analysis requires the solution of the parameter estimation problem hundreds of times (by default:1000) thus the overall computational cost may increase rapidly.
  • Plots/reports confidence or uncertainty regions by pairs of unknowns (cloud of solutions + ellipses) *Plots/reports confidence or uncertainty for individual unknonwns (histograms)

In order to use AMIGO_RIdent you need to specify upper and lower bounds on the parameters.

inputs.PEsol.id_global_theta='all';
inputs.PEsol.global_theta_max=[1 1 1 1 1];
inputs.PEsol.global_theta_min= [0 0 0 0 0];
inputs.PEsol.global_theta_guess = 0.5*(inputs.PEsol.global_theta_max +  inputs.PEsol.global_theta_min);

inputs.nlpsol.eSS.maxeval=5000;
inputs.nlpsol.eSS.local.solver='dn2fb';

You need to choose a suitable number of optimizations for the robust identifiability analysis. Default is 500. The value of 10 used here serves the illustration purpose of this tutorial only.

inputs.rid.conf_ntrials=5;

More information regarding the inputs used in this example can be found here.

AMIGO_Prep(inputs);
AMIGO_RIdent(inputs);
